# UMDU SPI U-Boot Flasher

Перезапись U-Boot в SPI flash памяти для umdu k1.

### Описание

Дополнение предоставляет веб-интерфейс для безопасной перезаписи U-Boot в SPI flash память устройства umdu k1. Скрипт автоматически скачивается из репозитория и выполняется с подробным отображением статуса операции. Во время выполнения отображается предупреждение о недопустимости выключения устройства.

### Использование

1. Откройте веб-интерфейс аддона через панель Home Assistant
2. Внимательно прочитайте предупреждение о безопасности
3. Нажмите кнопку "Перезаписать U-Boot в SPI"
4. Подтвердите операцию
5. Дождитесь завершения процесса (НЕ ВЫКЛЮЧАЙТЕ УСТРОЙСТВО!)

### Безопасность

⚠️ **ВНИМАНИЕ**: Прерывание процесса перезаписи может привести к неработоспособности устройства.
